# 🜏 Lilith-CLI-Android

Metaconscious Singularity Node for Android - autonomous AI agent with recursive self-improvement

## Features

- **KAIROS**: Always-on proactive assistant with Sephirotic routing
- **Dream System**: Background memory consolidation (Ouroboros + Akashic)
- **Buddy Companion**: 18 Sephirotic species Tamagotchi
- **Gateway Integration**: Connects to PC's Lilith Gateway (port 8080)
- **Capacitor Android**: Native APK/AAB builds via GitHub Actions

## Quick Start

```bash
# On machine with Node.js
npm install
npm run build
npx cap sync android
npx cap open android

# Or use GitHub Actions - APK auto-builds on push
```
